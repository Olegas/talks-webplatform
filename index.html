Геолокация

Позволяет определить местоположение пользователя.

Работает не только на мобильных устройствах, где есть GPS, но и даже в Desktop-браузерах!

http://caniuse.com/#feat=geolocation

Определение координат…

Battery Status API

Позволяет отслеживать уровень заряда аккумулятор устройства

http://caniuse.com/#feat=battery-status

Live Demo!

Online/offline Status API

Событие при смене статуса коннективности.

http://caniuse.com/#feat=online-status

Тэг <video>

Позволяет внедрять проигрыватель видео без дополнительных плагинов (например без Flash)

http://caniuse.com/#feat=video

Тэг <video>

getUserMedia/Stream API

Доступ к устройствам пользователя:

  • Веб-камера
  • Микрофон

http://caniuse.com/#feat=stream

Еще демо

  • http://shinydemos.com/facekat/
  • http://webaudiodemos.appspot.com/

WebRTC

RTC — Real Time Communication. Прямые соединения между клиентами, минуя сервер. Peer-2-Peer прямо из браузера!

http://caniuse.com/#feat=rtcpeerconnection

WebRTC. Применение.

  • Общение
  • Игры

File API

Доступ к сождержимому файлов, выбранных через input[type=file] без сервера!

http://caniuse.com/#feat=fileapi

File API

WebGL

3D-графика, управляемая на JavaScript!

Шейдеры, текстуры, вершины — вот это вот все. Прямо у вас в браузере!

http://caniuse.com/#feat=webgl

WebGL Demo

  • Chrome Experiments
  • http://madebyevan.com/webgl-water/

WebWorkers / ServiceWorkers

  • Выполнение процессов (например длинных синхронных вычислений) в фоне
  • Proxy-сервер, обработка запросов/подмена результатов, кэширование

http://caniuse.com/webworkers, http://caniuse.com/serviceworkers
